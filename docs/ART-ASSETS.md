# Visual asset pieces

The first prototype keeps the cabinet art as separate editable SVG pieces instead of hiding the look inside one generated scene.

- `public/assets/cabinet-art.svg` — front fascia and side-panel language
- `public/assets/pegboard-art.svg` — rear peg-board graphic
- `public/assets/coin-face.svg` — YES coin face
- `public/assets/coin-back.svg` — YF coin reverse
- `public/assets/showcase-panel.svg` — active-player trinket-wall layout reference
- `public/assets/yf-logo-source.png` — supplied Yokefellow source logo

These files establish one shared palette: dark navy, violet, cyan, green, gold, and restrained graffiti lines. Future skin/toy/trinket families should be designed against this cabinet rather than independently inventing unrelated styles.
